# VCP CLI Privacy Policy

VCP CLI runs locally and only sends data required for the AI task you initiate.

## Data Handling

- Code and command context may be sent to your configured model provider when required.
- Commands execute in your local environment.
- This fork does not claim ownership of your repository content.
