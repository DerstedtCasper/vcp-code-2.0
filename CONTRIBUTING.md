# Contributing to VCP CLI

Thanks for contributing to VCP Code.

## Scope

- This repository is maintained as a long-term fork focused on VCP Code and VS Code extension delivery.
- When documentation conflicts with runtime behavior, fix code/docs together and keep them consistent.

## Development

```bash
bun install
bun run --cwd packages/kilo-vscode check-types
bun run --cwd packages/kilo-vscode lint
```

## Pull Requests

- Keep PRs focused and reviewable.
- Include test/build results in PR description.
- For UI changes, include screenshots (before/after).
