# VCP Code Community Code of Conduct

We are committed to an open, respectful, and inclusive community.

## Our Standards

- Be respectful and constructive.
- Focus on technical collaboration, not personal attacks.
- Welcome contributors with different backgrounds and experience.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

## Reporting

If you experience unacceptable behavior, report it via:
https://github.com/DerstedtCasper/vcp-code-2.0/issues
